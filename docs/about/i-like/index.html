Απολαμβάνω

Εδώ μπορείτε να βρείτε μερικά πράγματα που μου άρεσαν αρκετά ώστε να γραφτούν εδώ. Κατ' ακρίβεια πρόκειται για τις αγαπημένες μου επιλογές για τα διάφορα είδη που αντιπροσωπεύουν.

Πολλά από αυτά τα διάβασα/είδα/μπλα από προτάσεις σε ιστοσελίδες άλλων, γι'αυτό είπα να κάνω και εγώ κάτι παρόμοιο στην ιστοσελίδα μου. "Τα ωραία πράγματα ανήκουν σε όλους" είπε κάποιος. Νομίζω.

Here they come.

Τραγούδια

Κάποια στιγμή θα συνειδητοποιήσω πόσα πολλά τραγούδια αδικώ με τον σύντομο κατάλογο που ακολουθεί και θα τον βγάλω τελείως. Μέχρι τότε, αναφέρω εδώ επιλεκτικά μερικά που θυμάμαι να έμειναν στο playlist (με repeat on) για πολλές μέρες. Κυρίως για να έχω κάτι να θυμάμαι άμα αναπολώ παλιές εποχές.. Κλισέ ξεκλισέ, φλωρέ ξεφλωρέ, κάποια στιγμή, κάπως, για κάποιο λόγο έτυχε να τα αγαπήσω.
