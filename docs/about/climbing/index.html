Αναρρίχηση

Κάποια στιγμή εδώ θα υπάρξει αναλυτικό αφιέρωμα στο άθλημα που αφιερώνω αρκετές ώρες της βδομάδας μου, την αναρρίχηση βράχου.

Μέχρι τότε, μπορείτε να πάρετε μια ισχνή γεύση από τις σχετικές φωτογραφίες που έχω αναρτήσει. Ελπίζω κάποια στιγμή να έχω και κάποιο βίντεο να δημοσιεύσω. Το οποίο αν έπρεπε να κάνω κάποια πρόβλεψη θα έλεγα σε λίγες βδομάδες.

Αν ήμουν εσύ, δεν θα στηριζόμουν στο τελευταίο.
