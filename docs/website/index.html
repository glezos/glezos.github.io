Λογότυπο ιστοχώρου

Μια και το ζήτησαν κάποιοι φίλοι, ετοίμασα το λογότυπο του ιστοχώρου και σε μεγαλύτερες αναλύσεις από 16x16 pixels για χρήση π.χ. σε εικονίδια σε RSS aggregators κλπ. Μπορείτε να το χρησιμοποιήσετε ελεύθερα όπου θέλετε για να συνδέσετε σ' αυτή την ιστοσελίδα.

Αν χρειάζεστε το λογότυπο σε διανυσματική μορφή (π.χ. σε SVG format), μη διστάσετε να μου στείλετε ένα e-mail.
