Επικοινωνία

Μπορείτε να επικοινωνήσετε μαζί μου στο dimitris *AT* glezos *DOT* com (αντικαθιστώντας τα '*AT*' και '*DOT*' με '@' και '.' αντίστοιχα).

Αν για κάποιο λόγο θέλετε να κωδικοποιήσετε το μήνυμα που θα μου στείλετε ώστε να το διαβάσω εγώ και κανένας άλλος ενδιάμεσα, μπορείτε να χρησιμοποιήσετε το GPG/PGP Public Key μου.

Για τους φαν του IM, το id μου στο Jabber δίκτυο είναι glezos@jabber.org (το subbie@jabber.org μπορεί να θεωρηθεί επισήμως παρωχημένο).

Μπορεί να τύχει να με βρείτε και στο IRC στα δίκτυα Freenode ή GRnet με το nickname 'glezos' ή 'subbie'.
