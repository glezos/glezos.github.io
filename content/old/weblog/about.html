Πού βρίσκομαι;

Το weblog αυτό είναι μια συλλογή από σκέψεις, νέα και γνώμες σχετικά με τους υπολογιστές (και όχι μόνο) από εμένα. Είναι ένας χώρος για να γράφω για την τεχνολογία, το web και τη ζωή, μια αποθήκη αποθήκη απόψεων, φωτογραφιών, ιδεών και άλλων κομματιών που συμπληρώνουν το πάζλ.

Δώστε ένα περίπατο, ίσως σας ενδιαφέρει κάτι ;-)

Weblog?

..αν μπορεί κανείς να το χαρακτηρίσει έτσι.. Μπορείτε να το δείτε σαν τη προσωπική μου εφημερίδα, το τετράδιο με τα σκίτσα μου, ένα περιεκτικό sniffing του τι σκέφτομαι και πιστεύω, αλλά προς Θεού, αυτό δεν είναι ημερολόγιο. Τα θέματα που πραγματεύεται ξεκινούν από προγραμματισμό και σχολιασμό των δρώμενων στον κόσμο της πληροφορικής, μέχρι ειδήσεις, πολιτισμό και μερικές φορές φιλοσοφία.

Chit-Chat = Κουβεντούλα
Tic-Tac-Toe = Τρίλιζα
Chit-Chat + Tic-Tac-Toe = Chit-Chat-Choe. Weblog με κουβεντούλα, σκέψεις και ομορφούλι όνομα.

Περί του weblog

Μπορώ να σχολιάσω κάτι που έγραψες;

Σαφώς. Σ' αυτό το weblog μπορεί κανείς να σχολιάσει αυτά που γράφω ελεύθερα. Για προφανείς λόγους, τα σχόλια πρέπει να είναι ακολουθούν κάποιους λογικούς κανόνες συμπεριφοράς, τους οποίους μπορείτε να δείτε παρακάτω, στο comment policy.

Υπάρχουν και post που δεν μπορώ να σχολιάσω!

Η δυνατότητα σχολιασμού απενεργοποιείται κυρίως σε παλιές καταχωρήσεις γιατί είναι ανώφελο να σχολιάσει κανείς παλιές καταχωρήσεις και για να κρατείται η συνοχή στο περιεχόμενο των σχολίων, η οποία όσο περνάει ο καιρός, φθείρεται. Επίσης κάποιες καταχωρήσεις έχουν σκοπό ενημέρωσης, παρά συζήτησης - και εκεί μπορεί να είναι απενεργοποιημένη η δυνατότητα σχολιασμού.

Τι χρησιμοποιείς για το site;

Το weblog χρησιμοποιεί την Movable Type για τη διαχείρηση των post/comments/κλπ η οποία είναι γραμμένη σε perl. Το positioning γίνεται με CSS, χωρίς τη χρήση tables. Το μεγαλύτερο μέρος του site, το οποίο είναι τα posts, τα σχόλια και οι επιμέρους σελίδες όπως αυτή που διαβάζετε τώρα χρησιμοποιούν γραμματοσειρά Georgia. Tα υπόλοιπα κομματάκια, όπως το logo, o footer των post, το σώμα των μπλοκ στο αριστερό μέρος της κεντρικής σελίδας απεικονίζονται με Verdana. Αν το σύστημα σας δεν έχει αυτές τις γραμματοσειρές, χρησιμοποιείται η γενικότερη οικογένεια γραμματοσειρών sans-serif.

Πώς κάνω link σε κάποιο άρθρο;

Αν θέλετε να κάνετε link σε κάποια καταχώρηση αντί σε όλο τον ιστοχώρο, μπορείτε να το κάνετε μέσω του "Permanent Link" tag που υπάρχει στο κάτω μέρος της κάθε καταχώρησης - π.χ. κάνοντας κλικ πάνω του και αντιγράφοντας το URL που θα ανοίξει στο Location bar του browser σας (θα είναι κάτι του στυλ /weblog/archives/000021.html).

Έχετε υπ'όψιν σας ότι το υλικό σ'αυτό τον ιστοχώρο είναι κατοχυρωμένο κάτω από μια Creative Commons Licence.

Υπάρχει κάποιο RSS feed (XML)?

Σαφώς! Οι τελευταίες 10 καταχωρήσεις υπάρχουν σε RSS 1.0 (.rdf) μορφή και σε RSS 2.0 (xml). Μπορείτε να τα κάνετε ότι θέλετε - π.χ. να τα δείτε μέσα από κάποιο RSS Aggregator ή να τα τσιμπάτε όποτε θέλετε.

Αν σκοπεύετε να τα χρησιμοποιήσετε για κάτι όμορφο, θα χαρώ πολύ να το μάθω ;-)

Comment policy

Θέλω να σχολιάσω κάτι. Πρέπει να προσέξω κάτι;

Κάποια από τις καταχωρήσεις επιτρέπουν την αποστολή σχολίων από τους αναγνώστες. Δεν χρειάζεται registration και τα σχόλια δημοσιοποιούνται αμέσως. Mην χρησιμοποιήσετε HTML στα σχόλια: Τα URI που γράφετε γίνονται αυτόματα links, τα line breaks και οι παραγράφοι δημιουργούνται κανονικά.

Σκοπός αυτού του χαρακτηριστικού, είναι η συζήτηση. Παρ'όλη τη δημοκρατία, αυτό δεν παύει να είναι η προσωπική μου ιστοσελίδα. Τα σχόλια που κρίνω ότι είναι ανάρμοστα ή προσβλητικά θα γίνονται moderated.

Πιο συγκεκριμένα θα διαγράφονται:

  • Σχόλια που περιέχουν flames/trolling ή προσβλητικά σχόλια. Επίσης, χρήστες που προσβάλλουν εμένα ή την οικογένεια μου θα γίνονται blacklisted με όλα τα δυνατά μέσα.
  • Σχόλια που περιέχουν άσχετα links σε εμπορικές σελίδες θα διαγράφονται αμέσως και οι χρήστες που τα υπέβαλαν θα γίνονται blacklisted.
  • Σχόλια που είναι εκτός θέματος, μια και αντιβαίνουν τον ορισμό του 'σχόλιου'. Αν θέλετε να πείτε κάτι που δεν έχει σχέση με ένα άρθρο, απλά μην το πείτε στο thread της καταχώρησης. Αν κάποια καταχώρηση δεν επιτρέπει την αποστολή σχολίων αλλά εσείς θέλετε να γράψετε κάτι, μην το γράψετε στο thread άλλης καταχώρησης γιατί θα είναι off-topic.
  • Σε κάθε περίπτωση, μπορείτε να μου στείλετε ένα mail.

Με δυο λόγια...;

Επoικοδομητική συζήτηση. Ανταλλαγή απόψεων. Νιώστε ελεύθερα να πείτε τη γνώμη σας, αλλά μείνετε στο θέμα και σεβαστείτε τους άλλους συγγραφείς και αναγνώστες. Τα προφανή δηλαδή.

Αλήθεια, γιατί να νοιάζεται κανείς για το τι πιστεύεις;
