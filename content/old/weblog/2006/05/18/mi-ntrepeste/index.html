    • Τι διάλο δεν πάει καλά με αυτούς τους Άγγλους; Πριν απο λίγο καιρό ήταν οι πινακίδες στα αυτοκίνητα. Χτες είδα στο Slashdot οτι σκεύτονται να κάνουν παράνομη την χρήση και την διανομή network security tools που μπορούν να χρησημοποιηθούν από hackers.

      Πίσω στον νόμο για την παράδωση των κλειδιών…

      Σήμερα δεν υπάρχει χώρος για αυτόν τον νομο. Αφότου γίνει το έγκλημα, το δικαστήριο μπορεί να ζητήσει το κλειδί και έπειτα από παρουσίαση στοιχειων. Όπως το ένταλμα ερευνας στο σπίτι ενός υποπτου.

      Εκτος και αν η κυβέρνηση είναι διατεθειμένη να αποκρυπτογραφεί όλα τα μηνύματα και να έχει ένα σεβαστό αριθμό υπαλλήλων να τα κοιτάει ένα-ένα, τα εγκλήματα δεν πρόκειται να αποτραπούν. Ένα αυτόματο υπολογιστικό σύστημα δεν είναι αρκετό διότι οι εγκληματίες ποτε δεν επικοινωνούν κυριολεκτικά στις τηλεπικοινωνίες τους. Ένας βομβιστής θα αναφερθεί στην βόμβα με μια άλλη λέξη, όπως “το γλυκό”.

      Ο σημερινός μηχανισμός επαρκεί για την δικαιοσύνη. Το αίτημα για όλα τα ιδωτικά κλειδιά δεν έχει να κάνει με την δικαιοσύνη, αλλα με την απληστία. Απληστία εξουσίας. Είναι ακριβώς το ίδιο πράγμα με το να ζητάει η κυβέρνηση το κλειδί του σπιτιού σου για να μπει μέσα όταν λείπεις και να κάνει ερευνα σε περίπτωση που είσαι βομβιστής η παιδόφιλος. Δεν μπορώ να πω με σιγουριά εξαρχής πως αυτό είναι σωστό η λάθος. Για να βρούμε τι πρέπει να κάνουμε, πρέπει πρώτα να αναρωτηθούμε τι είδους κοινωνία θέλουμε. Πάντοτε υπάρχουν κακά πράγματα που θα συμβαίνουν όσο και να προσπαθούμε να τα αποτρέψουμε. το θέμα είναι ποσα θα θυσιάσουμε στην προσπάθεια μας να αγγίξουμε την ψευδαίσθηση. Η κοινωνία σαν σύνολο έχει να κερδίσει περισσότερα από την ιδιωτικότητα (privacy) από ότι να συλλάβει μερικούς εγκληματίες.

      Αντί να ζητάνε να δουν τι λένε οι άνθρωποι, όφειλαν να το κάνουν δύσκολο σε κάποιον να βρει και να μεταφέρει εκρηκτικές ύλες. Με όλα όσα ξέρουμε, μια λαίμαργη κυβέρνηση θα μπορούσε να επιτρέπει βομβιστικές επιθέσεις μονο για να πείσει την κοινή γνώμη ότι ένα αίτημα σαν και αυτό είναι απαραίτητο (έχει ήδη συμβεί στο παρελθόν). Εάν μιλάμε για μακροχρόνια, τότε ξέρουμε πως εμείς (οι Ευρωπαίοι και Αμερικανοί) εισβάλαμε στα σπίτια αυτών των ανθρώπων, και τώρα πληρώνουμε το τίμημα.

      Οι άνθρωποι τελικά θυσιάζουν την ελευθερία για χάρη του αισθήματος τις ασφάλειας. Είναι μόνο ένα εικονικό φάρμακο (placebo). Κανένας δεν πρόκειται να είναι πιο ασφαλής, απλά τώρα θα κινδυνεύουμε και από τις κυβερνήσεις.

      Μου ήρθε στο μυαλό μια σχετική φράση που διάβασα κάποτε, “Ο κόσμος είναι μια κωμωδία για αυτούς που σκέφτονται, και μια τραγωδία για αυτούς που αισθάνονται”. Ας γελάσουμε καλύτερα :)

      Υ.Γ.: αυτό ίσως βοηθήσει κάπως: Truecrypt
