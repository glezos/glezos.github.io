  • Counting submissions through Transifex in the past month… >>> import datetime >>> aMonthAgo = datetime.datetime.now() - datetime.timedelta(days=31) >>> ActionLog.select(AND( ActionLog.q.type=="submit", ActionLog.q.timestamp > aMonthAgo)).count() 1281
