μαρκαδόρος + τσίχλα = τσiχλαδόρος (απλός ή ανεξήτηλος)
ψαλίδι + τσίχλα = τσιχλίδι
πένα + τσίχλα = πέχλα
bic + τσίχλα = μπίχλα
