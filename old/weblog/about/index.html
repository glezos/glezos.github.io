Το weblog αυτό

…à la FAQ

Πού βρίσκομαι;

Το weblog αυτό είναι μια συλλογή από σκέψεις, νέα και γνώμες σχετικά με τους υπολογιστές (και όχι μόνο) από εμένα. Είναι ένας χώρος για να γράφω για την τεχνολογία, το web και τη ζωή, μια αποθήκη απόψεων, φωτογραφιών, ιδεών και άλλων κομματιών που συμπληρώνουν το πάζλ.

Δώστε ένα περίπατο, ίσως σας ενδιαφέρει κάτι ;-)

Weblog?

..αν μπορεί κανείς να το χαρακτηρίσει έτσι.. Μπορείτε να το δείτε σαν τη προσωπική μου εφημερίδα, το τετράδιο με τα σκίτσα μου, ένα περιεκτικό sniffing του τι σκέφτομαι και πιστεύω, αλλά προς Θεού, αυτό δεν είναι ημερολόγιο! Τα θέματα που πραγματεύεται ξεκινούν από προγραμματισμό και σχολιασμό των δρώμενων στον κόσμο της πληροφορικής, μέχρι ειδήσεις, πολιτισμό και μερικές φορές φιλοσοφία.

Περί του weblog

Μπορώ να σχολιάσω κάτι που έγραψες;

Σαφώς! Σ’ αυτό το weblog μπορεί κανείς να σχολιάσει αυτά που γράφω ελεύθερα. Παρακαλώ διαβάστε το comment policy πριν σχολιάσετε.

Υπάρχουν και post που δεν μπορώ να σχολιάσω!

Η δυνατότητα σχολιασμού απενεργοποιείται κυρίως σε παλιές καταχωρήσεις γιατί είναι ανώφελο να σχολιάσει κανείς παλιές καταχωρήσεις και για να κρατείται η συνοχή στο περιεχόμενο των σχολίων, η οποία όσο περνάει ο καιρός, φθείρεται. Επίσης κάποιες καταχωρήσεις έχουν σκοπό ενημέρωσης, παρά συζήτησης – και εκεί μπορεί να είναι απενεργοποιημένη η δυνατότητα σχολιασμού.

Τι χρησιμοποιείς για το site;

Το weblog χρησιμοποιεί την Movable Type το WordPress για τη διαχείριση των post/comments/κλπ η οποία είναι γραμμένη σε perl php. Το positioning γίνεται με CSS, χωρίς τη χρήση tables. Το μεγαλύτερο μέρος του ιστοχώρου απεικονίζονται με γραμματοσειρά Verdana, ενώ οι καταχωρήσεις και τα σχόλια εμφανίζονται με γραμματοσειρά Georgia.

Επίσης, χρησιμοποιούνται κάποια plugins του wordpress, όπως μεταξύ άλλων το Comment live preview του Ιάκωβου Κωνσταντίνου. Τα mini-posts εμφανίζονται με μια τροποποιημένη έκδοση του αντίστοιχου plugin του Morgan Doocy.

Πώς κάνω link σε κάποιο άρθρο;

Αν θέλετε να κάνετε link σε κάποια καταχώρηση αντί σε όλο τον ιστοχώρο, μπορείτε να το κάνετε μέσω του μόνιμου συνδέσμου που υπάρχει στο κάτω μέρος της κάθε καταχώρησης – π.χ. κάνοντας κλικ πάνω του και αντιγράφοντας το URL που θα ανοίξει στο Location bar του browser σας (θα είναι κάτι του στυλ /weblog/2004/07/07/barbara-hendricks/).

Παρακαλώ έχετε υπ’όψιν σας ότι το υλικό σ’αυτό τον ιστοχώρο είναι κατοχυρωμένο κάτω από μια Creative Commons Licence.

Υπάρχει κάποιο RSS feed (XML)?

Φυσικά! Οι τελευταίες 10 καταχωρήσεις υπάρχουν σε RSS 1.0 μορφή, σε RSS 2.0 και Atom. Μπορείτε να τα κάνετε ότι θέλετε – π.χ. να τα δείτε μέσα από κάποιο RSS Aggregator ή να τα τσιμπάτε όποτε θέλετε.

Αν σκοπεύετε να τα χρησιμοποιήσετε για κάτι όμορφο, θα χαρώ πολύ να το μάθω ;-)

Mini-posts

Mini-posts; Τι είν’ πάλι τούτο;

Όπως λέει και ο Matt, η μορφή ενός weblog υπαγορεύει το γράψιμο του. Η κάθε καταχώρηση του weblog μου έχει ένα μεγάλο τίτλο, μια ημερομηνία, είναι αρχειοθετημένη σε διάφορες κατηγορίες και άλλα καλούδια. Αυτό είναι σκόπιμο: Με αυτό τον τρόπο, αναγκάζομαι να δίνω ιδιαίτερη έμφαση και σκέψη στο περιεχόμενο μιας καταχώρησης ώστε να τα αξίζει τα πυροτεχνήματα.

Ωστόσο, αρκετές φορές θέλω να γράψω για πράγματα τα οποία δεν αξίζουν τα πυροτεχνήματα. Από μικρές σημειώσεις, links σε ειδήσεις και καταχωρήσεις άλλων ιστολόγων, σχόλια και σκόρπιες σκέψεις. Πράγματα τα οποία μπορεί να μην ενδιαφέρουν τον καθένα.

Γι’αυτό το σκοπό δημιούργησα μια ειδική κατηγορία καταχωρήσεων, τα mini-posts ώστε να μπαίνουν εκεί όλα αυτά. Κάθε mini-post έχει δικό του μόνιμο σύνδεσμο και επιτρέπει το σχολιασμό. Τέλος, τα mini-posts παρέχονται (φυσικά) και σε RSS/Atom μορφή.

Πολιτική σχολίων

Θέλω να σχολιάσω κάτι. Πρέπει να προσέξω κάτι;

Οι περισσότερες από τις καταχωρήσεις επιτρέπουν το σχολιασμό από τους αναγνώστες. Σκοπός αυτού του χαρακτηριστικού, είναι η έκφραση απόψεων, η συζήτηση και η πολυφωνία, στοιχεία που κάνουν αξιόλογο τούτο δω το ιστολόγιο.

Παρ’ όλη τη δημοκρατία, αυτό δεν παύει να είναι η προσωπική μου ιστοσελίδα. Σχόλια τα οποία κρίνω ότι είναι ανάρμοστα ή προσβλητικά θα επισημειώνονται (annotated). Αυτά συμπεριλαμβάνουν flames, trolling, ad hominem επιθέσεις σε εμένα ή κάποιο άλλο σχολιαστή και άσχετα links σε εμπορικές σελίδες. Επίσης θα επισημειώνονται σχόλια τα οποία είναι εκτός θέματος, μια και αντιβαίνουν τον ορισμό του ‘σχόλιου’.

Μια και είμαι κάθετα εναντίων της λογοκρισίας, θα ακολουθήσω την τακτική των Sam Ruby και Mark Pilgrim: Tο κείμενο δεν θα διαγράφεται από τον ιστοχώρο αλλά θα επισημειώνεται (annotated) με οπτική διαγραφή και θα συνδέεται σε αυτή την παράγραφο. Για παράδειγμα, ένα σχόλιο σαν το:

Αυτή είναι η άποψη μου κι επίσης είσαι πιο χοντρός κι από ελέφαντα.

θα επισημειώνεται ως εξής:

Αυτή είναι η άποψη μου κι επίσης είσαι πιο χοντρός κι από ελέφαντα.

Με δυο λόγια…;

Επoικοδομητική συζήτηση, ανταλλαγή απόψεων. Νιώστε ελεύθερα να πείτε τη γνώμη σας, αλλά μείνετε στο θέμα και σεβαστείτε τους άλλους συγγραφείς και αναγνώστες. Τα προφανή δηλαδή.

Για οτιδήποτε θέλετε να μου πείτε, εντός ή εκτός θέματος, μπορείτε ανά πάσα στιγμή να μου στείλετε ένα mail.

Μπορώ να χρησιμοποιήσω HTML στο σχόλιο μου;

Μπορείτε να χρησιμοποιήσετε τα εξής XHTML tags στο σχόλιο σας (τα attributes που αναφέρονται είναι υποχρεωτικά): <?php echo allowed_tags(); ?>.

Αναλυτικά:

  • a για συνδέσμους (links), img για εικόνες
  • em, strong για έμφαση
  • ul, li για λίστες (bullets)
  • blockquote και q (inline) για παραθέσεις (π.χ. λόγια άλλων, απαντήσεις κλπ), code για είσοδο/έξοδο υπολογιστή (fixed-width)

Παρακαλώ δώστε ιδιαίτερη προσοχή στη σωστή χρήση των tags βάσει του XHTML προτύπου. Ο ιστοχώρος αυτός κάνει ό,τι μπορεί για να είναι προσβάσιμος από άτομα με αναπηρίες χρησιμοποιώντας ορθή XHTML. Αυτό σημαίνει πως τα tags υποχρεωτικά πρέπει να κλείνουν και να είναι γραμμένα με μικρά γράμματα και τα attributes πρέπει να βρίσκονται μέσα σε εισαγωγικά.
